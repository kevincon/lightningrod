Lightning Rod is a Java-based Dropbox client that connects your flash drive to the cloud.

About

Dropbox is great...when you have access to their desktop client. But you can't use the desktop client on public computers in labs, libraries, and internet cafes because you're not given admin privileges to install software.

Dropbox does have a terrific website that lets you download individual files (or multiple files as a zip archive), but your changes aren't synced until you re-upload the file to the website. Also, Dropbox's website doesn't let you upload entire folders, so it can be a real hassle to work with Dropbox files on a public computer.

We're building a Dropbox client from scratch that can run from a flash drive. We call it "Lightning Rod."

FAQ

  1. What is Lightning Rod?

    Lightning Rod is a Dropbox client for flash drives.

  2. Wait, doesn't Dropbox already have a client?

    If you are using a computer that has restrictions on installing programs,(like in a computer lab, library, or internet cafe), you won't be allowed to install Dropbox's desktop client.

    That's where Lightning Rod comes in. Lightning Rod allows you to change/upload/download your Dropbox files on your flash drive, and any changes you make will be synced automatically.

  3. Will this work on all computers I take my flash drive with Lightning Rod to?

    Yes! Lightning Rod was designed to use Java, which most computers have (Macs even come with it already installed). If you do not have the latest version of Java, you can get it here.

  4. Is there any registration required?

    All you need is a Dropbox account.

  5. How is LightningRod different from DropboxPortableAHK?

    We believe Lightning Rod is the first cross-platform portable Dropbox client, using Java and your web browser to access files.

    It requires no previous installation of the Dropbox client on a portable drive.
